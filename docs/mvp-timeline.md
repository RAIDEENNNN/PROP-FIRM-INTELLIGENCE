# Precise MVP Timeline

This timeline assumes one focused developer/designer working with external accounts ready when needed.

## Week 1 — Launchable intelligence website

Day 1:

- Finalize logo, design system and homepage.
- Build responsive header/mobile navigation.
- Complete directory, profile, compare and pricing pages.

Day 2:

- Build spread matrix UI.
- Add forex, metals/commodities, indices, crypto and synthetic sample universe.
- Add source/plugin status page.

Day 3:

- Implement backend auth routes.
- Add user/trader profile routes.
- Add Prisma schema and seed script.

Day 4:

- Add firm, compare, spreads, news, alerts, reviews and admin API routes.
- Add Stripe-ready billing routes.

Day 5:

- Mobile QA.
- SEO metadata.
- Legal placeholders upgraded to professional draft.
- Build validation and Netlify preview deploy.

## Week 2 — Real production wiring

Day 6:

- Create hosted Postgres database.
- Run migrations and seed.
- Deploy API backend.

Day 7:

- Connect frontend to backend auth.
- Store JWTs safely.
- Build dashboard saved firms and alerts UI.

Day 8:

- Connect Stripe products, checkout and billing portal.
- Configure webhook signature verification.

Day 9:

- Connect email provider for verification/password reset.
- Connect news provider.
- Connect first spread provider.

Day 10:

- End-to-end testing.
- Analytics/error monitoring.
- Soft launch to 10–25 traders for feedback.

## Week 3 — Trust and monetization

- Add review moderation UI.
- Add payout proof upload/storage.
- Add affiliate link click tracking.
- Add featured listing workflow.
- Add first business report/export.

## Week 4 — Growth

- Publish SEO pages.
- Add Google Search Console.
- Add onboarding survey.
- Add recommendation tuning.
- Start outbound to prop firm affiliates and trader communities.
